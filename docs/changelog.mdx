## v2.0.0 (2023-06-15)

### Breaking Changes

- Removed support for legacy API endpoints
- Changed authentication method from API keys to OAuth 2.0

### New Features

- Introduced real-time collaboration features
- Added dark mode support across all platforms

### Improvements

- Enhanced search functionality with fuzzy matching
- Optimized database queries for faster response times

### Bug Fixes

- Fixed issue with user profile updates not saving correctly
- Resolved intermittent connection errors on mobile devices

## v1.5.0 (2023-05-01)

### New Features

- Redesigned dashboard for better user experience

### Bug Fixes

- Fixed pagination issues in the activity log
- Resolved compatibility issues with older browser versions
